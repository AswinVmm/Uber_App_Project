# Uber_App_Project
A functional Uber clone project with features like rider and driver dashboard ,payment gateways and map routing using APIs like Leaflet,ORS,Stripe , use of socket.io for data tansfer and firebase for database 
#How to run:-
change the directory, cd Backend and run the command:npx nodemon
open another terminal and change directory, cd uber_clone and run the command:npm run dev
